# Python Flask CI/CD Lab

This is a Python Flask application designed for a CI/CD (Continuous Integration/Continuous Deployment) lab. The project demonstrates how to set up and use GitHub Actions for automated testing and deployment workflows.

## Installing Requirements

To install the required dependencies, run:

```bash
pip install -r requirements.txt
```

## Running Tests

To run the tests, use pytest:

```bash
python -m pytest
```

This will discover and run all tests in the `tests/` directory.
